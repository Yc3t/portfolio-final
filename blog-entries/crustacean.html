How to create a macro

macro_rules! print {

    ($($arg:tt)*) => {{ // MATCH ZERO OR MORE TT, AND BIND EACH ONE TO THE VARIABLE ARG

        $crate::io::_print($crate::format_args!($($arg)*));

    }};

}

$ -> CAPTURE THE NEXT THING : -> Tells the macro designator what to expect (eg. EXPECT A TOKEN TREE(tt))

* -> Tells the macro to expect zero or more repetitions => -> Separate the macro pattern from the body

$crate -> variable that refers to the current crate

:: -> Access items in a module (io module in this example)
